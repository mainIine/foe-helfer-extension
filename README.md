## Forge Hammer
##### A fork of the FoE Helper Browser Extension

Forge Hammer is an extension for the browser game Forge of Empires that displays ingame information in a structured way, gathers some player- and guild-related data over time, and can help out with daily tasks and guild management.

Feel free to add documentation to our wiki, help out with translations via the ingame module or create a pull request after discussing your idea in a ticket.
