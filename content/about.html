Forge Hammer

Free browser addon for Forge of Empires
maintained by outoftheline and Beelzebob666

This looks familar?!

This addon is based on the widely known FoE Helper, an open-source project under the GNU Affero General Public License v3.0 that we have contributed to for many years.

Why not continue working on FoE Helper? It boils down to not feeling appreciated - especially in regards to creative differences, aswell as different perspectives about the use of AI.

As Forge Hammer mostly uses the same variables, it is not recommended to run both addons at the same time.

Automation?

There will never be anything added that breaks the game rules. Anything included from FoE Helper is tolerated by InnoGames thus far.

Translations

We can only provide English and German translations ourselves. If you want to help with other languages, look at the language settings. We added a translation tool and a description on how to use it over there. Thank you!
